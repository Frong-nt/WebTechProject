"สิ่งมีชีวิตใต้ท้องทะเลที่มีเปลือก และหนาม"

8. ไฟลัมเอไคโนเตอร์มาตา

โครงสร้างของไฟลัมเอไคโนเตอร์มาตา

พบเจอได้เฉพาะใต้ทะเลเท่านั้น


    สิ่งมีชีวิตในไฟลัมเอไคโนเดอร์มาตา (Echinodermata) จะสามารถพบได้แค่ใต้ทะเลเท่านั้น เช่น ดาวทะเล เม่นทะเล และปลิงทะเล ซึ่งสิ่งมีชีวิต เหล่านี้จะมีผิวที่ขรุขระ และบางชนิดจะมีหนามแทนผิวที่ขรุขระ ซึ่งเป็นอันตรายต่อสิ่งมีชีวิตอื่นๆ


เป็นสิ่งมีชีวิตที่สมมาตรแบบรัศมี


    จากการที่สิ่งมีชีวิตในกลุ่มนี้มีเมเทมอร์โฟซิส (Metemofosis) และมีการแบ่งลำตัวในช่วงโตเต็มวัยเป็น 5 ส่วน หรือทวีคูณของ 5 ยื่นออกมาจากแกนกลมที่เป็นศูนย์กลาง ทำให้ระบบประสาทเป็นแบบวงแหวน และแยกแขนง ออกไปตามแขนทั้ง 5 ข้าง ซึ่งตัวอย่างที่เห็นได้ชัดเจน คือ ดาวทะเล ที่มีแขนยื่นออกมาจากแกนกลางลำตัวออกมา 5 ข้าง เมื่อมีการแบ่งสมมาตรจะได้สมมาตรแบบรัศมี (Radially symmetrical) ในขณะที่ดาวทะเลในระยะตัวอ่อนจะมีสมมาตรแบบครึ่งซีก (Bilaterally symmetrical)


สามารถสืบพันธุ์ได้ทั้งแบบอาศัยเพศ และไม่อาศัยเพศ


    สิ่งมีชีวิตในกลุ่มนี้เมื่อมีการขาดของส่วนของแขนใดแขนหนึ่ง ส่วนที่ขาดจะสามารถเติบโตไปเป็นตัวเต็มอีกตัวหนึ่ง ซึ่งเป็นการสืบพันธุ์แบบไม่อาศัยเพศ ในขณะที่การสืบพันธุ์จะเป็นแบบอาศัยเพศด้วยการปฏิสนธิภายนอก


ลักษณะสำคัญของสัตว์พวกเอไคโนเดอร์มาตา


    1. ผนังลำตัว เป็นผิวหยาบขรุขระ เพราะมีสารสะสมอยู่
    2. ลำตัวไม่มีหัวท้ายเด่นชัด อยู่ในทะเลทั้งสิ้น
    3. มีส่วนยื่นออกจากจุดศูนย์กลางในแนวรัศมีเป็น 5 แฉกหรือ 5 เท่าของแฉก
    4. มีระบบเลือดแบบเปิด และหายใจโดยใช้เหงือก (Gill papillae) หรือ Cloacal respiratory tree ในปลิงทะเล
    5. สมมาตรแบบรัศมี แต่ตัวอ่อนเป็นแบบเหมือนกัน 2 ซีก
    6. การสืบพันธุ์แบบใช้เพศ และไม่ใช้เพศ
    7. เป็นสัตว์ทะเลทั้งสิ้น ไม่พบในแหล่งน้ำจืด
    8. มีระบบท่อน้ำ (Water vascular system) ภายในร่างกายซึ่งเจริญมาจากช่องตัวในระยะตัวอ่อน ภายในท่อบรรจุด้วยน้ำเค็มจากภายนอก ลักษณะภายนอกของระบบนี้ที่พอเห็นได้ คือ เท้าท่อ (Tube feet) เมื่อทำงานร่วมกันทำให้สามารถเคลื่อนไหว จับอาหาร หายใจ และรับความรู้สึกได้ ระบบนี้ถือว่าเป็นระบบ ไฮดรอลิก (Hydraulic system) ซึ่งไม่มีในสัตว์ไฟลัมอื่น
    9. ทางเดินอาหารเป็นแบบสมบูรณ์ ยกเว้นสมาชิกใน Class Ophiuroidea ทางเดินอาหารจะไม่มีทวารหนัก เช่น ดาวเปราะ
    10. ระบบหมุนเวียนโลหิต มีลักษณะลดลงไปอย่างมาก บางชนิดไม่มีเลย ส่วนการขับถ่ายไม่มีอวัยวะขับถ่ายที่ทำหน้าที่โดยตรง
    11. ระบบประสาทไม่มีส่วนสมองที่แท้จริง แต่พบว่ามีระบบประสาทวงแหวน (Nerve ring) ล้อมรอบหลอดอาหารไว้ และมีเส้นประสาทรัศมี (Radial nerve) แยกออกจากประสาทวงแหวนไปเลี้ยงที่แขน อวัยวะรับความรู้สึกเจริญน้อยมาก


สัตว์ในไฟลัมนี้แบ่งออกเป็น 5 คลาส


    1. Class Asteroidea เช่น ดาวทะเล
    2. Class Ophiuroidea เช่น ดาวเปราะ
    3. Class Echinoidea เช่น หอยเม่น อีแปะทะเล
    4. Class Holothuroidea เช่น ปลิงทะเล
    5. Class Crinoidea เช่น พลับพลึงทะเล
